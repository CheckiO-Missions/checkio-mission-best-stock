Сперва вы можете принять за цену самой дорогой акции - ноль (0). Затем сравниваете цену первой акции с переменной, которой присвоили 0, а каждую последующую акцию - с предыдущей. Если новая акция окажется более дорогой - ее цену нужно занести в переменную, в которой хранится максимальная цена, а её название - в переменную, где хранится ответ (название самой дорогой акции).
Как мне по очереди сравнивать все цены акций?
Используйте цикл for
Мне нужно увидеть, как это выглядит в коде.
Решение этой задачи описанным способом выглядит так: 
def best_stock(data):    
    max_price = 0
    answer = ''
    for s in data:
        if data[s] > max_price:
            max_price = data[s]
            answer = s
    return answer
