Перш за все, тобі потрібно дві змінні: одна для максимальної ціни, інша - для відповідного тікера (короткої назви) акції. Тоді, перебираючи всі значення в словнику, оновлювати значення обох змінних за умови, що ціна поточної акції вища за ту, яка зберігається у відповідній змінній. Після проходження всього словника, повернути значення тієї змінної, що зберігала тікер акції.
Але як порівняти всі ціни по черзі?
Використай для цього цикл for та вбудований метод словників items().
Мені потрібно побачити це у вигляді коду.
Розв'язок задачі з урахуванням попередніх рекомендацій матиме вигляд: 
def best_stock(data):    
    max_price = 0
    answer = ''
    for k, v in data.items():
        if v > max_price:
            max_price = v
            answer = k
    return answer
