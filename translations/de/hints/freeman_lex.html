def best_stock(data: dict[str, float]) -> str:

    max_price = 0
    answer = ''
    for k, v in data.items():
        if v> max_price:
            max_price = v
            answer = k

    return answer
Dies ist eine klare Schritt-für-Schritt-Lösung, bei der wir über das Wörterbuch iterieren, jeden Wert mit dem vorher deklarierten Wert vergleichen und einen größeren Preis und dessen Ticker speichern. 

def best_stock(data: dict[str, float]) -> str:

    return max(data, key=data.__getitem__)
    # or
    return max(data, key=lambda x: data[x])
Die Hinweislösung kann auch in dieser Form geschrieben werden. 

from functools import reduce


def best_stock(data):

    return reduce(lambda x,y: [x,y][y[1]> x[1]], data.items())[0]
Mit der vorimportierten Funktion reduce nehmen wir die ersten beiden Schlüssel-Wert-Paare, behalten die mit dem höheren Preis, nehmen das dritte Paar und vergleichen es mit dem vorherigen "Gewinner" und so weiter. Geben Sie den Ticker des verbleibenden Paares zurück.
