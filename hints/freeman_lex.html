def best_stock(data: dict[str, float]) -> str:

    max_price = 0
    answer = ''
    for k, v in data.items():
        if v > max_price:
            max_price = v
            answer = k

    return answer
This is a clear step-by-step solution, where we iterate over the dictionary, compare each value with pre-declared and store bigger price and its ticker. 

def best_stock(data: dict[str, float]) -> str:

    return max(data, key=data.__getitem__)
    # or
    return max(data, key=lambda x: data[x])
The hint solution may be written in these forms as well. 

from functools import reduce


def best_stock(data):

    return reduce(lambda x,y: [x,y][y[1] > x[1]], data.items())[0]
Using pre-imported reduce function we are taking first two key-value pairs, keep ones with bigger price, take third pair and compare with previous "winner" and so on. Return the ticker from the pair that remains.
